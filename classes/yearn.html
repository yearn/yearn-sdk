Class Yearn<T>

Yearn is a wrapper for all the services and interfaces of the SDK.

Yearn namespace can be instantiated as a class, providing configuration options that will then be used by all the services and interfaces:

import { Yearn } from "@yfi/sdk";

const provider = new JsonRpcProvider("http://localhost:8545");
const yearn = new Yearn(1, { provider });
